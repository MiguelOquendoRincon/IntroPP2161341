Archivos primer examen Introducción a la Computación Paralela

El código solicita un número inicial y uno final. Con esto crea un rango y
encuentra los números primos en dicho rango. El objetivo fue paralelizar 
la sección del código donde se realiza dicho proceso. Para esto se estrajo
la función encargada de encontrar si un número es primo o no y se paralelizó
el for que recorre el rango dado por el usuario.
